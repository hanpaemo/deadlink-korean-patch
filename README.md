# Deadlink 한글패치

**Deadlink** 비공식 한글패치 — 한패모 (hanpaemo)

## 설치 방법

1. [GitHub Releases](https://github.com/hanpaemo/deadlink-korean-patch/releases)에서 `zzz_deadlink_korean_P.pak` 다운로드
2. 게임 폴더에 복사:
   ```
   Steam\steamapps\common\Deadlink\Deadlink\Content\Paks   ```
3. Steam 실행 옵션 추가: `-culture=ko`
4. 게임 실행
